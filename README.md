# CUDA primeri

CUDA (C++) primeri sa vremenima potrebnim za izvršenje različitih kernela.

Dodatak:
  - AMD ROCm
  - Intel OneAPI

Potrebno:
  - nVidia kartica
  - CUDA toolkit
  - nvcc kompilator

Kompilacija:
  - Linux: nvcc naziv.cu -o naziv
  - Windows: nvcc naziv.cu -o naziv.exe

Pokretanje:
  - Linux: ./naziv
  - Windows: naziv.exe
